# 🎯 人像LOGO智能体 — 人物肖像品牌字标Logo生成系统

> **一句话**：上传你的照片 + 填写品牌信息 → 一键生成可商用的个人肖像品牌Logo提示词，支持中英文输出，适配主流AI绘图工具

---

## 📌 项目简介

这是一个基于AI提示词工程设计的**人物肖像品牌字标Logo生成系统**。不同于普通的头像转插画工具，本系统专注于输出**完整、可商用、有设计师感**的个人品牌Logo方案，将真实人物肖像转化为品牌人格符号。

### 核心能力

| 能力项 | 说明 |
|--------|------|
| 🎯 肖像Logo生成 | 基于真实照片，保留人物识别特征，输出Logo化视觉提示词 |
| 🏷️ 完整结构 | 肖像层 + 字标层 + 徽章层 三层整合，非简单拼接 |
| 🌍 中英双语 | 默认输出中文提示词，可选英文，适配国内外AI绘图工具 |
| 🎨 多行业适配 | 摄影/咖啡/健身/餐饮/设计/咨询/医疗/自媒体等全覆盖 |
| 💼 品牌完成度 | 成品可直接用于头像/名片/门头/社交媒体/周边产品 |
| 📦 独立可分享 | 单个SKILL.md文件，放入任何AI工具即用 |

## 🎨 Logo设计原理

### 三层结构体系

```
┌─────────────────────────────┐
│      徽章层 (Badge)         │  ← 半圆底托/圆形框/丝带/横线 整合包装
├─────────────────────────────┤
│      字标层 (Wordmark)      │  ← 品牌名定制字标，行业气质
├─────────────────────────────┤
│      肖像层 (Portrait)      │  ← 基于照片的块面化/版画化艺术肖像
└─────────────────────────────┘
```

### 核心设计原则

1. **肖像是记忆锚点** — 保留真实人物识别特征，Logo化处理，非写实复刻
2. **字标是品牌识别** — 定制字标不能是普通排字，必须有行业气质和设计变化
3. **职业属性可感知** — 通过道具（相机/咖啡杯/哑铃等）和字体风格传递行业定位
4. **三层是一个整体** — 肖像+文字+徽章结构融为一体，不是上下机械拼接
5. **配色克制有记忆点** — 黑白灰主体 + 1个强调色，无廉价渐变

## 📋 参数说明

生成Logo时需要提供以下参数：

| 参数 | 示例 | 说明 |
|------|------|------|
| 📷 人物照片 | 上传 | 必须上传，作为肖像创作依据 |
| 品牌名/人名 | 硬核KK | 核心识别，Logo主视觉字 |
| 副标题 | AI自学室 | 行业/品类定位语（选填） |
| 类型/行业 | 摄影工作室、咖啡主理人 | 决定整体风格和道具方向 |
| 品牌定位 | 专业、亲民、高端 | 影响构图和色彩策略 |
| 人物身份 | 社群发起人、设计师 | 职业特征，决定道具和姿态 |
| 职业属性 | 主理人感、手艺感 | 影响字标风格选择 |
| 情绪气质 | 专业稳重、温暖亲切 | 决定整体调性 |
| 字标风格 | 现代理性、复古主理人 | 直接影响品牌气质 |
| 主色调 | 深蓝色、黑灰色 | 品牌主色，用于强调色 |
| 辅助色 | 低饱和度棕红 | 点缀色，小面积使用 |
| 画幅比例 | 1:1 | 1:1适合头像，4:5适合竖版 |

## ❓ 常见问题

**Q: 生成的Logo可以直接商用吗？**
A: 提示词生成的图片需遵守对应AI工具的服务条款。建议作为设计参考，最终商用建议进行专业设计优化。

**Q: 必须上传真人照片吗？**
A: 是的，本系统的核心价值在于将真人肖像转化为品牌符号，照片是必需的输入。没有照片可先用类似人物特征的照片替代测试。

**Q: 如何获得更好的生成效果？**
A: 1) 选择正面或半侧面照片，光线均匀；2) 参数描述越具体越好；3) 选择与个人气质匹配的情绪词；4) 1:1画幅适合头像，4:5适合品牌展示。

**Q: 支持哪些行业？**
A: 覆盖摄影、咖啡、健身、餐饮、设计、咨询、医疗、教育、自媒体、潮流等主流个人品牌行业。模板之外的其他行业直接填入即可。

**Q: 和普通头像转卡通有什么不同？**
A: 本系统输出的是品牌Logo级的肖像处理（块面化/版画化/符号化），而非普通卡通插画。最终成品具备品牌感、设计师感和商业完成度。
